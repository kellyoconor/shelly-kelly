# Message Workflow - MANDATORY STEPS

## Before EVERY message to Kelly:

### Step 1: Context Check (ALWAYS)
```bash
python3 /data/workspace/scripts/smart-context-check.py
```

### Step 2: Review Output
- ✅ Safe to ask about topics marked as "Can ask"

Use context data to inform conversation, not assumptions

### Step 4: Send Message
No activity questions without data verification
